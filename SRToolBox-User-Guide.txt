SRToolBox 申瑞工具箱
==================

下载地址
--------

访问Releases页面下载最新版本：
https://github.com/Verherder/SRToolBox-Releases/releases

下载以下文件，下载最新版本：
- SRToolBox.exe - 应用程序
- setup_env.bat - 环境配置脚本
- requirements.txt - 依赖列表
- PDFtoPrinter.exe - PDF打印程序（必需）
- SRToolBox-User-Guide.txt - 本文件


环境配置
--------

方式一：使用 setup_env.bat 自动配置（推荐）

1. 将下载的所有文件放到同一目录
2. 双击运行 setup_env.bat
3. 脚本会自动：
   - 检测/安装 Python 3.10
   - 升级 pip
   - 创建虚拟环境 D:\venv
   - 安装所有依赖包


方式二：手动配置

1. 安装 Python 3.10（从 python.org 下载安装包）

2. 创建虚拟环境
   python -m venv D:\venv

3. 激活虚拟环境
   D:\venv\Scripts\activate.bat

4. 安装依赖
   pip install -r requirements.txt


运行应用
--------

双击 exe 运行即可。

首次运行会在 exe 同目录下生成 config.yaml 配置文件，
用于保存用户设置的打印机、输出目录等选项。

重要：config.yaml 保存了您的个人设置，删除后将无法恢复。
如果误删，请重新下载 exe 并双击运行，程序会自动创建新的配置文件。


功能模块
--------

1. 报关资料批量打印
   将报关单、提单、发票等 PDF 文件批量打印。


2. 库存周转报表
   根据以下数据源生成库存周转报表：
   - FBA仓库存
   - 本地仓库
   - 海外仓
   - AWD库存
   - 采购单待到货


常见问题
--------

Q: 运行报错 "DLL load failed"
A: 需要安装 Visual C++ Redistributable
   下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe

Q: 报错 "qpdf##.dll not found" 或类似错误
A: PDFtoPrinter 需要 x86 版本的最新版的 VC Redistributable，
   下载地址：https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist
   选择 x86（非 x64）版本安装

Q: pip 安装失败
A: 检查网络连接，或尝试切换其他镜像源

Q: 找不到 Python
A: 确保按setup_env.bat成功创建venv环境
